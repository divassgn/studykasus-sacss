Visi & Misi Program Studi

Visi

Menjadi program studi informatika terdepan yang menghasilkan lulusan berkualitas tinggi dan berdaya saing global dalam bidang teknologi informasi dan komputer.

Misi
  • Menyelenggarakan pendidikan informatika berkualitas tinggi
  • Mengembangkan penelitian inovatif di bidang teknologi
  • Melakukan pengabdian masyarakat melalui teknologi
  • Membangun kerjasama dengan industri dan institusi global
Kurikulum Informatika
  • Algoritma & Struktur Data
  • Pemrograman Web & Mobile
  • Database Management System
  • Artificial Intelligence
  • Machine Learning
  • Cybersecurity
  • Software Engineering
  • Computer Networks
Prospek Karir
  • Software Developer
  • Data Scientist
  • AI/ML Engineer
  • Cybersecurity Specialist
  • Mobile App Developer
  • DevOps Engineer
  • System Analyst
  • Tech Entrepreneur
Keunggulan Program Studi Informatika
  • Laboratorium Modern: Fasilitas lab dengan teknologi terkini
  • Dosen Berkualitas: Pengajar berpengalaman industri dan akademis
  • Kurikulum Up-to-date: Sesuai perkembangan teknologi industri
  • Project-Based Learning: Pembelajaran berbasis proyek nyata
  • Industry Partnership: Kerjasama dengan perusahaan teknologi
  • International Exposure: Program pertukaran mahasiswa
  • Research Opportunities: Kesempatan penelitian tingkat nasional
  • Career Support: Bimbingan karir dan job placement
